# Project Overview
The Migraene_app is designed to assist users in tracking and managing their migraine conditions. It offers features for logging migraine occurrences, triggers, and relief methods.

# Setup Instructions
To set up the Migraene_app locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/schalui/Migraene_app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Migraene_app
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

# Architecture Guide
The architecture of the Migraene_app is based on a modular structure, ensuring scalability and maintainability. The main components include:
- **Frontend**: The user interface built with React.
- **Backend**: Handling data processing and storage using Node.js and Express.
- **Database**: MongoDB is used for persistent data storage.

Ensure to follow best practices while developing and adding new features to maintain the architecture's integrity.
